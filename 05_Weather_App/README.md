# Weather App

## Description
A weather forecasting application built with React and Vite. Integrates with a weather API to display real-time weather data for different locations.

## How to Run
1. Open a terminal and navigate to the `frontend` folder (if applicable) or the root directory.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
4. Open the provided localhost URL in your browser.
